# 📊 Superstore Sales Analysis using SQL

## Project Overview

This project analyzes the Sample Superstore dataset using SQL to identify business insights related to sales, profit, customer segments, products, shipping modes, and regional performance.

The project demonstrates SQL skills commonly required for entry-level Data Analyst roles, including data exploration, aggregation, filtering, window functions, subqueries, Common Table Expressions (CTEs), views, and business analysis.

---

## Dataset

- **Dataset:** Sample Superstore
- **Total Records:** 9,994
- **Database:** MySQL
- **File Format:** CSV

---

## Objective

The objective of this project is to answer real business questions using SQL, such as:

- Which states generate the highest sales?
- Which categories are the most profitable?
- Which customer segment contributes the most revenue?
- Which shipping mode has the highest average profit?
- Does offering higher discounts reduce profitability?
- Which products are generating losses?
- Which cities perform best within each region?
